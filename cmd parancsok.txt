1) Mappa létrehozása
mkdir A13C-login-session
cd A13C-login-session

2) Node projekt inicializálás
npm init -y

3) Függőségek
npm i express express-session bcrypt better-sqlite3 dotenv

 4) (opcionális) fejlesztői eszköz
npm i -D nodemon

5) Könyvtárak
mkdir public
mkdir data

 6) Fájlok létrehozása (ha nincs kódszerkesztőd, a notepaddal is megnyithatod)
 A tartalmakat lejjebb találsz: egyszerűen cseréld a fájlokban.
notepad .env
notepad package.json
notepad server.js
notepad db.js
notepad public\index.html
notepad public\login.html
notepad public\register.html
notepad public\profile.html
notepad public\style.css
notepad public\utils.js

7) nodemon-nal kényelmes:
npx nodemon server.js
 
vagy simán:
node server.js
