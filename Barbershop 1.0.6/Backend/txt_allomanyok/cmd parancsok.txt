1) Mappa létrehozása
mkdir BARBERSHOP
cd BARBERSHOP

2) Node projekt inicializálás
npm init -y

3) Függőségek
npm i bcrypt dotenv express express-session jsonwebtoken mariadb nodemon sequelize multer nodemailer

4) (opcionális) fejlesztői eszköz
npm i -D nodemon

5) Könyvtárak
IMPORTANT: mkdir Backend - ez után lépj bele ebbe a mappába a: "cd Backend" paranccsal
mkdir public
mkdir config
mkdir src
mkdir src\contorllers
mkdir src\models
mkdir src\routes
